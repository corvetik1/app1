# AccountCards.js

## Описание
`AccountCards.js` — React-компонент, отображающий карточки счетов (дебетовые/кредитные) с их балансами и действиями.

## Основные функции
- Отображение списка счетов в виде карточек.
- Показ текущего баланса и основных данных.
- Кнопки для редактирования/удаления счетов.

## Зависимости
- **React**: Для UI.
- **@mui/material**: Компоненты Material-UI.
- **react-redux**: Для Redux.
- **../store/financeActions**: Действия для финансов.
- **../utils/logger**: Логирование.

## Структура
- **Состояние**: Данные счетов из Redux.
- **Обработчики**: Редактирование/удаление счетов.
- **UI**: Карточки с информацией о счетах.

## Использование
Используется в `FinancePage.js` для отображения счетов.
```jsx
import AccountCards from './AccountCards';
<AccountCards />
