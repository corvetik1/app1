# ReportModal.js

## Описание
`ReportModal.js` — модальный компонент для отображения отчёта по тендерам с возможностью фильтрации, сортировки и экспорта.

## Основные функции
- Отображение отфильтрованных тендеров в таблице.
- Фильтрация по этапам и поиск по тексту.
- Перетаскивание строк для изменения порядка.
- Экспорт данных в Excel.

## Зависимости
- **React**: Для UI.
- **@mui/material**: Компоненты Material-UI.
- **react-beautiful-dnd**: Для перетаскивания.
- **xlsx**: Работа с Excel.
- **file-saver**: Сохранение файлов.
- **react-redux**: Для Redux.
- **../store/tenderActions**: Действия для тендеров.
- **../utils/logger**: Логирование.

## Структура
- **Состояние**: Локальное для фильтров, поиска, подсветки строк.
- **Обработчики**: Фильтрация, сортировка, экспорт.
- **UI**: Модальное окно с таблицей.

## Использование
Используется в `TendersPage.js` как модальное окно отчёта.
```jsx
import ReportModal from './ReportModal';
<ReportModal open={open} onClose={handleClose} />
