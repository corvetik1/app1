# TenderRow.js

## Описание
`TenderRow.js` — React-компонент, представляющий одну строку таблицы тендеров. Используется внутри `TenderTable.js` для отображения и редактирования данных тендера.

## Основные функции
- Отображение данных тендера в ячейках.
- Редактирование ячеек (текст, числа, выбор из списка).
- Выбор цвета метки через всплывающее окно.

## Зависимости
- **React**: Для UI.
- **@mui/material**: Компоненты Material-UI.
- **@mui/x-date-pickers**: Для `DatePicker`.
- **react-redux**: `useDispatch`.
- **../store/tenderActions**: Действия для тендеров.
- **../utils/logger**: Логирование.

## Структура
- **Состояние**: Локальное для управления `Popover` цвета.
- **Обработчики**: `handleCellChange`, `handleColorLabelSelect`, фильтрация колонок.
- **UI**: Строка таблицы с ячейками.

## Использование
Используется внутри `TenderTable.js` как часть рендеринга строк. Не предназначен для самостоятельного использования.
```jsx
<TenderRow tender={tender} ... />
